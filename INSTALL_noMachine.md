# Installing noMachine

## On both Local and Client

Download the Package
```html
https://download.nomachine.com/download/?id=1&platform=linux
```
Select the Linux 64-bit .deb package.
```bash
cd ~/Downloads
sudo dpkg -i nomachine_*.deb
```
Once installed, search for "NoMachine" in your applications menu to start the program.   On both computers.

????
Connecting to Your Ubuntu Machine
    Locate IP: Open NoMachine on the Ubuntu machine to find its local IP address on the welcome panel.
    Connect: On the client computer, enter the IP address, port (usually 4000), and protocol (NX) to connect.
    Credentials: You will need your Ubuntu username and password to log in. 

Note: You may need to enable "Accept connections" in the NoMachine settings (server preferences) to allow incoming requests.
