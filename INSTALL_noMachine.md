# Installing noMachine

## On both Local and Client

Download the Package
```html
https://download.nomachine.com/download/?id=1&platform=linux
```
Select the Linux 64-bit .deb package.
```bash
cd ~/Downloads
sudo dpkg -i nomachine_*.deb
```
Once installed, search for "NoMachine" in your applications menu to start the program.   On both computers.

As running for first time, simply use all the defaults.

Also check 'do not show' boxes
