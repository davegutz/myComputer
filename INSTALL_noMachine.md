# Installing noMachine

For Linux, Download the Package https://download.nomachine.com/download/?id=1&platform=linux
Select the Linux 64-bit 
```bash
deb package
cd ~/Downloads
sudo dpkg -i nomachine_*.deb
```
For windows, download the Package https://download.nomachine.com/download/?id=3&platform=windows
Install using usual windows process

Once installed, search for "NoMachine" in your applications menu to start the program.   On all machines.  As running for first time, simply use all the defaults.

Also check 'do not show' boxes

Connect all to network as davegutz@alum.mit.edu.  Bitwarden has the password.
Select blank for 'Start the system on server startup' for security
